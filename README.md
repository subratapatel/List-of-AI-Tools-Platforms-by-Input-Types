List of AI Tools & Platforms by Input Type

This repository contains a curated list of popular AI tools and platforms, categorized based on the type of input they process, including Text, Image, Audio, Video, and Coding.

The objective of this assignment is to understand and document how modern AI systems handle different data modalities and are applied across various domains such as content creation, software development, media processing, and automation.

The datasets are provided in CSV and Excel formats for easy access, analysis, and future extension.

Categories included:

Text-based AI tools

Image generation and editing tools

Audio and voice-based AI tools

Video generation and editing tools

AI tools for coding and software development
